Ejemplo

"El LED se enciende si algo se acerca"

🎯 Objetivo:
Detectar la presencia de un objeto a menos de 10 cm usando un sensor de proximidad, y encender un LED como señal de alerta.

👣 Instrucciones:

  • Arma el circuito en la protoboard siguiendo las conexiones.
  • Carga el código en el Arduino desde el IDE.
  • Abre el Monitor Serial para ver la distancia medida.

Coloca tu mano u objeto frente al sensor:

  • Si está a menos de 10 cm, el LED se enciende.
  • Si está a más de 10 cm, el LED se apaga.

🧰 Materiales:

  • 1 Arduino Uno
  • 1 sensor ultrasónico HC-SR04
  • 1 LED
  • 1 resistencia de 220Ω
  • Cables jumper
  • Protoboard

Conexiones

Sensor HC-SR04:

  • VCC → 5V de Arduino
  • GND → GND
  • Trig → Pin 9
  • Echo → Pin 10

LED:

  • Ánodo (pata larga) → Resistencia de 220Ω → Pin 7
  • Cátodo (pata corta) → GND

Código básico

const int trigPin = 9;
const int echoPin = 10;
const int ledPin = 7;
long duracion;
int distancia;


void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
}


void loop() {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);


  duracion = pulseIn(echoPin, HIGH);
  distancia = duracion * 0.034 / 2;


  Serial.println(distancia);


  if (distancia < 10) {
    digitalWrite(ledPin, HIGH); // Se enciende el LED si algo está cerca
  } else {
    digitalWrite(ledPin, LOW);  // LED apagado si no hay objeto cerca
  }


  delay(500);
}
