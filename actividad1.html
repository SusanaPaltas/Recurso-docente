Actividad

Actividad para mover servo entre 0°, 45 °, 90°y 180°.

  • Desarrollar el circuito virtual con un servomotor en Tinkercad y programarlo para moverse entre 0°, 45 °, 90°y 180°.
  • Se deberá modificar el código para crear una secuencia personalizada de movimientos así como añadir comentarios al código para explicar su funcionamiento.
  • Subir la captura de la simulación, el código fuente y el link de donde se encuentra realizado, además de la evidencia del movimiento del servo en los angulos especificados.

Esquema de referencia 

Código fuente:

#include <Servo.h>

Servo servo1; // Primer servomotor
Servo servo2; // Segundo servomotor

void setup() {
servo1.attach(6); // Primer servo al pin 6
servo2.attach(5); // Segundo servo al pin 5
}

void loop() {
// Subida de ángulos
servo1.write(0);
servo2.write(0);
delay(1000);

servo1.write(45);
servo2.write(45);
delay(1000);

servo1.write(90);
servo2.write(90);
delay(1000);

servo1.write(180);
servo2.write(180);
delay(1000);

// Bajada de ángulos
servo1.write(90);
servo2.write(90);
delay(1000);

servo1.write(45);
servo2.write(45);
delay(1000);

servo1.write(0);
servo2.write(0);
delay(1000);
}
