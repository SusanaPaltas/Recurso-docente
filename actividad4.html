Actividad

“Alarma de proximidad con sonido”

🎯 Objetivo de la actividad:
Diseñar e implementar un sistema de alarma sencillo que emite un sonido mediante un zumbador y enciende un LED como alerta visual cuando un objeto se aproxima al sensor de proximidad.

🧰 Materiales necesarios:
1 placa Arduino Uno
1 zumbador (buzzer activo o pasivo)
1 sensor de proximidad (ultrasónico HC-SR04 o infrarrojo)
1 LED
1 resistencia de 220Ω (para el LED)
Protoboard
Cables RCA o jumpers

Conexiones del circuito

🔊 Zumbador:
Pin positivo del zumbador → pin 9 del Arduino
Pin negativo del zumbador → GND


👁️ Sensor de proximidad (ej. sensor infrarrojo):
VCC → 5V del Arduino
GND → GND del Arduino
OUT → pin 2 del Arduino


💡 LED de aviso:
Ánodo del LED (pata larga) → resistencia de 220Ω → pin 13 del Arduino
Cátodo (pata corta) → GND

👨‍🔧Instrucciones paso a paso:

Paso 1

Arma el circuito en la protoboard, siguiendo cuidadosamente las conexiones indicadas. Asegúrate de que todos los componentes estén bien colocados y los cables firmemente conectados.

Paso 2

Abre el entorno Arduino IDE en tu computadora. 

Paso 3

Escribe el siguiente código básico:

const int sensorPin = 2;     // Pin de salida del sensor
const int buzzerPin = 9;     // Zumbador
const int ledPin = 13;       // LED de alerta


void setup() {
  pinMode(sensorPin, INPUT);
  pinMode(buzzerPin, OUTPUT);
  pinMode(ledPin, OUTPUT);
}


void loop() {
  int estadoSensor = digitalRead(sensorPin);


  if (estadoSensor == HIGH) {      // Objeto detectado
    digitalWrite(buzzerPin, HIGH); // Suena el zumbador
    digitalWrite(ledPin, HIGH);    // Enciende el LED
  } else {
    digitalWrite(buzzerPin, LOW);  // Apaga el zumbador
    digitalWrite(ledPin, LOW);     // Apaga el LED
  }


  delay(100);
}

Paso 4

Sube el código a tu placa Arduino y observa cómo actúa el sistema.

Paso 5

Prueba acercando un objeto al sensor. Si todo está correcto:

  • El zumbador emitirá un sonido.
  • El LED se encenderá como señal visual.

Implementa lo siguiente: 

  • Cambiar la distancia de detección.
  • Agregar un segundo LED que parpadee como alerta secundaria.
  • Incluir un mensaje en el monitor serial que indique cuándo hay detección.
